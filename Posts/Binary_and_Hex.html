Binary

Binary (Base 2) is a number system using 1’s and 0’s. These 1’s and 0’s can be used to signify switch or light positions, 1 is on and 0 is off. The first computers used binary to program them and the result was displayed as a series of lights. In computing a binary digit is known as a bit, 8 bits make a byte. a byte is the basic unit of storage in computing. In one byte there are 255 possible combinations of binary numbers. Hexadecimal is used in computing because it allows more information to be stored in fewer digits and it is easy to convert between binary and Hexadecimal. Hexadecimal is commonly seen when defining colours in web pages using pairs of hex numbers to define the Red, Green and Blue parts of a colour. In computing, we put a hash(#) to signify a hex number. #ffffff is white, meaning every colour and #000000 is black using no colour! Using this system will give you 16,581,375 possible colours!

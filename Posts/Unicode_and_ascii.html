Unicode and Ascii

My first look at Unicode and Ascii.

Ascii

Ascii is a 7 bit data type. Therefore, it can have a maximum of 128 characters. This is 2^7 characters.


Unicode

Unicode is an 8 bit data type. It can have a maximum of 256 characters. This is 2^8. Unicode has more characters because it is “Universal”, so it has to have every symbol in every language, while ascii is american so that only has the english/ american alphabet.
